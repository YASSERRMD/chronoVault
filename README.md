# ChronoVault

Time-based contract obligation tracking system.

## Quick Start

```bash
# Start with Docker
docker-compose up -d

# Or run locally
cd backend && go run cmd/server/main.go
cd frontend && npm install && npm run dev
```

## Tech Stack

- Frontend: Vue 3 + Vite
- Backend: Go + Gin
- Database: SQLite
- Real-time: WebSocket
